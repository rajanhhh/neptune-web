    Spacing Utilities

    Several utility classes are available for spacing, using .{property}-{sides}-{size} naming.

    Property options

    • m ‚Äì for classes that set margin
    • p ‚Äì for classes that set padding

    Sides

    • a ‚Äì for classes that set the margin or padding on all sides of the element
    • t ‚Äì for classes that set the top side of the property
    • r ‚Äì for classes that set the right side of the property
    • b ‚Äì for classes that set the bottom side of the property
    • l ‚Äì for classes that set the left side of the property
    • x ‚Äì for classes that set both left and right
    • y ‚Äì for classes that set both top and bottom

    Sizes

    • 0 ‚Äì for classes that eliminate the margin or padding by setting it to 0
    • 1 ‚Äì for classes that set the margin or padding to space-8 (8px)
    • 2 ‚Äì for classes that set the margin or padding to space-16 (16px)
    • 3 ‚Äì for classes that set the margin or padding to space-24 (24px)
    • auto ‚Äì for classes that set the margin to auto

    And only for vertical cases:

    • 4 ‚Äì for classes that set the margin-top, margin-bottom or padding-top, padding-bottom to space-32 (32px)
    • 5 ‚Äì for classes that set the margin-top, margin-bottom or padding-top, padding-bottom to space-40 (40px)

    Responsive sections

    These are the sizes of larger, responsive spacing used for separating sections of a page:

    • section-1 classes provide 16px, 24px or 32px of spacing depending on screen width
    • section-2 classes provide 32px, 48px or 64px of spacing depending on screen width
    • section-3 classes provide 48px, 72px or 96px of spacing depending on screen width
