    By default NavigationOption is meant to be used with Icons.

    If you want to use full colour & spot illustrations instead then set showMediaCircle to false.

    By default icons & illustrations are hidden on small/mobile screens. If you want them to be visible then set showMediaAtAllSizes to true.
